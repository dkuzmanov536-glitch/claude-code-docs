Започни сега Официална документация
Агентски инструмент за писане на код, който чете кода ти, редактира файлове, изпълнява команди и се интегрира с инструментите ти за разработка.

Какво е Claude Code?

Claude Code е AI асистент за програмиране, който помага да изграждаш функции, да оправяш бъгове и да автоматизираш задачи при разработка. Разбира целия ти проект и може да работи едновременно с множество файлове и инструменти.

Какво можеш да правиш

Автоматизиране на задачи

claude "write tests for the auth module, run them, and fix any failures"
bash

Git commits и pull requests

claude "commit my changes with a descriptive message"
bash

CLI автоматизация

# Анализ на логове
tail -200 app.log | claude -p "Slack me if you see any anomalies"

# Автоматични преводи в CI
claude -p "translate new strings into French and raise a PR for review"

# Преглед на промени за сигурност
git diff main --name-only | claude -p "review these changed files for security issues"
bash

CLAUDE.md — Персонализация

Добави CLAUDE.md файл в корена на проекта — Claude го чете при всяка сесия. Използвай го за стандарти за код, архитектурни решения и предпочитани библиотеки.

Свързване с MCP

Чрез Model Context Protocol — Claude Code може да чете Google Drive, да обновява Jira, да дърпа данни от Slack и много повече.
